# FinalprojectofDoom
Challenges- 
1.) The very first challenge I encountered was that pygame didn't work with the version of python that I had. And I wasn't able to download the right version till I figured out I should download the older version (3.11) and then it worked.

2.) The first bug I encountered that took me the longest was after playing you wouldn't be able to close the game unless you force quit the application because of a misplaced continue statement. So I moved the event to the top of the loop so it continues running if you hit space or you can end it.

3.) I added more colors after the common red, green, and blue, but I couldn't figure out why the game still wouldn't work until I figured out that it was because I had to add more keystrokes to the colors. 

Learned-
How to use pygame, user inputs to game events, track time and reaction speed, how to use GitHub.

Enjoyed-
Seeing the project actual work was a great feeling, and fixing bugs while seeing what actually changes when I fix them was satisfying.
